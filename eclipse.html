Shortcut key

Posted on June 25, 2018

We develop with Java using Eclipse. I have learned some useful shortcuts from web site and friends. I would like to introduce them I often use.

Shortcut key

Shortcut key Action
Ctrl+F11 Run
Ctrl+Space Open Content Assist
Ctrl+Shift+/ Add comment(/* */)
Ctrl+Shift+\ Remove Comment(/* */)
Ctrl+/ Comment or remove comment(//)
Ctrl+Shift+f Format
Alt+Shift+R Rename identifier

I hope these can help you. If you know other useful shortcuts. Let's share!
