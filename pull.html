How to update local repository from remote repository

After Wednesday, if remote master branch was updated, please update your local repository.

Explanation with SourceTree

  1. change to master branch
    2. First, checkout the master branch from your develop branch. You can checkout by dubble click 'master'.

    When you checkout branch, if you have this kind of error, you have to commit your changes or stash them before you switch branches.

    how to stash

  2. pull origin master branch
    3. Then, pull master branch.

  3. checkout develop branch
    4. checkout develop branch again.

  4. merge master branch
    5. merge master branch to develop branch.
